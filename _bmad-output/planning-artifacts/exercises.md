# Exercises - Reusable Exercise Library

This document contains detailed descriptions of exercises for reuse in lessons across chapters.

---

## Exercise 1: Harmonic Interval Audiation

**Objective**: Develop audiation by imagining each note before singing

**Principle**: Listen → Focus on low note → Focus on high note → Sing → Verify

### Procedure

| Step | Action |
|------|--------|
| 1 | Play harmonic interval |
| 2 | "Focus on the low note" |
| 3 | Pause 2s |
| 4 | Play low note |
| 5 | Play same harmonic interval |
| 6 | "Focus on the high note" |
| 7 | Pause 2s |
| 8 | Play high note |
| 9 | Play same harmonic interval |
| 10 | "Now, sing the interval" |
| 11 | Display success color |
| 12 | Play melodic interval |

### Parameters

| Parameter | Options |
|-----------|---------|
| Starting notes | C, F, G, D, A |
| Intervals | Octave, Fifth, Major third |

## Exercise 2: Melodic Identification (Alternation)

**Objective**: Recognize a melodic interval

**Principle**: Play note1 - note2 - note1 (low/high/low) or note1 - note2 - note1 (high/low/high)

### Parameters

| Parameter | Options |
|-----------|---------|
| Alternation type | Low-High-Low (L-H-L) or High-Low-High (H-L-H) |
| Starting notes | C, F, G, D, A |
| Target interval | Unison, Octave, Fifth, Major third |

### Examples

| Notes | Pattern | Answer |
|-------|---------|--------|
| C - G - C | L-H-L | Fifth |
| C - C (octave) - C | L-H-L | Octave |
| G - D - G | L-H-L | Fifth |
| D - D (octave) - D | H-L-H | Octave |

## Exercise 4: Introduction to Resolution

**Objective**: Feel the V → I resolution

**Principle**: Play a melody that stops on V, user completes by singing

### Parameters

| Parameter | Description |
|-----------|-------------|
| Melody | Major scale on 4 notes, stopped on 5th degree |
| Action | Sing the resolution (I) |

### Examples

| Melody played | User action |
|---------------|-------------|
| C - E - G - G | Sing C |
| F - A - C - C | Sing F |
| G - G - D - D | Sing G |

## Exercise 5: Descending Interval Singing

**Objective**: Descending interval singing

**Principle**: Start from a note, sing the descending interval then return

### Parameters

| Parameter | Options |
|-----------|---------|
| Starting notes | C, F, G, D, A |
| Intervals | Descending fifth, Descending third |

### Examples

| Given note | Expected singing |
|------------|------------------|
| G | G... C... G |
| D | D... G... D |
| E | E... C... E |

## Exercise 9: Interval in a Melody

**Objective**: Identify an interval in a melodic context

**Principle**: Listen to a short repeatable melody, note the missing note on a score

### Parameters

| Parameter | Description |
|-----------|-------------|
| Format | 2-3 note melody, stops before resolution |
| Score | Treble clef with starting note displayed |
| Repetition | Unlimited |

### Examples

| Melody played | Starting note | Action |
|---------------|---------------|--------|
| C - E - ... | C (treble clef) | Complete with G |
| F - A - ... | F | Complete with C |
